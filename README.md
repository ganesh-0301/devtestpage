# Travel Blog - Test Website

A comprehensive travel blog website featuring destinations across India, Nepal, Bhutan, and Dubai. This website is designed for testing visitor tracking scripts.

## Features

- **Multiple Destinations**: Travel stories from India, Nepal, Bhutan, and Dubai
- **Blog Posts**: Detailed travel articles with proper headings, images, and content
- **Gallery**: Visual gallery showcasing travel photos
- **Navigation**: Easy navigation by destination and category
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Structure

- `/app` - Next.js app directory with pages
- `/components` - Reusable React components
- `/lib` - Data and utility functions
- `/app/blog` - Blog listing and individual post pages
- `/app/gallery` - Photo gallery page

## Destinations

- **India**: Golden Triangle, Kerala Backwaters
- **Nepal**: Himalayan Trekking, Everest Base Camp
- **Bhutan**: Tiger's Nest, Punakha
- **Dubai**: Burj Khalifa, Desert Safari

## Technologies

- Next.js 14
